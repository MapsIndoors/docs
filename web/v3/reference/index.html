filter

Filter locations on the map by array of ids

Parameters

  • filter Array<string> An array of location ids to filter the map by.
  • fitView boolean Change the map viewport to fit the current filter.

LabelOptions

mapsindoors.LabelOptions

Properties

  • maxLength number Maximum numbers of charectors to display. If exceeded the label will be truncated and “…” will be added to the end.
  • pixelOffset google.maps.Size For offsetting the label.
  • style object
    • style.align string left, center, right
    • style.color string hex or rgba color
    • style.fontFamily string
    • style.fontSize string size in px
    • style.fontWeight string
    • style.strokeStyle string
    • style.strokeWeight number
    • style.shadowBlur number
    • style.shadowColor string hex or rgba color
